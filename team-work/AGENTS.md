# Team Work Folder Rules

This folder contains isolated teammate workspaces.

- Each teammate agent should work only inside its selected `team-work/member-N/` folder.
- Each task must have a request brief, generated checklist, local review file, and approval record before becoming a leader proposal.
- Use `notes/learnings.md` for reusable discoveries that future teammate tasks should know.
- Do not treat team-work files as final product files until the leader integrates them.
