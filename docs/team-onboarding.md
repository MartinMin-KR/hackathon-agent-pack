# Team Onboarding

## What Teammates Need To Know

You do not need to know Git commands or code review.

You tell the agent what you want. The agent asks a few questions, makes the work clearer, builds or prepares it, checks it, and uploads the work.

## Setup

1. Log in to GitHub.
2. Open the project folder.
3. Run `bash scripts/install.sh`.
4. Choose your name from the list.
5. Open Codex in this folder.
6. Say what you want to work on.

## How Work Flows

The agent will:

1. Ask questions.
2. Research automatically.
3. Try a simulated customer interview using Korean personas.
4. Create a clear task spec.
5. Make the result.
6. Check it.
7. Show you how to inspect it.
8. Ask whether it matches what you wanted.
9. Send it to the leader only after you approve.

## Your Job

Your job is to judge whether the result feels right for your request. You do not need to judge whether the code is good.
