# Integration Proposal Rules

This folder contains teammate-approved proposals for leader review.

- A proposal must come from a completed local checklist and teammate-approved local review.
- The leader agent reviews proposals here and decides what moves into `product/`.
- Do not use this folder for scratch work.
- Every proposal should explain the intended product change, verification evidence, and known risk.
- Product-ready files for automatic integration belong under `files/product/`.
- The leader must mark `proposal.md` with `Accepted: yes` before running `integrate`.
